## 네임스페이스가 겹치는 퍼블릭 호스팅 영역과 프라이빗 호스팅 영역, 프라이빗 호스팅 영역 2개

Resolver에서는 가장 구체적인 일치를 기반으로 트래픽을 라우팅합니다.

```
seattle.accounting.example.com 조회 시

아래 2개의 영역 중 더 정확한 부분을 조회한다.
accounting.example.com (1순위)로 조회된다.
example.com (2순위)
```
