## Resolver

### Onpremis -> AWS DNS Query

(1) 내부망 -> VPC 까지 네트워크 오픈
(2) Route 53 Inbound Endpoint 생성 -> VPC 설정 및 IP 생성
(3) Rotue 53 Private hosted zone Record 작성
(4) 내부망 PC DNS (2)에서 생성한 IP 등록
(5) 조회

### Onpremis -> AWS DNS Query 기존 사용 DNS 활용 방법
